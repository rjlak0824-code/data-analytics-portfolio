# 🍔 FoodHub Order Demand Analysis

## Project Overview
FoodHub is a food aggregator app that allows customers in New York to order from multiple restaurants through a single platform. This project analyzes order data to uncover demand patterns across restaurants, cuisines, and delivery times — enabling data-driven decisions to improve business operations and customer experience.

## Business Problem
With the growing number of restaurants and increasing demand for online food delivery, FoodHub needs to understand:
- Which restaurants and cuisines are most popular?
- What factors influence customer ratings and delivery times?
- When are peak ordering times?
- How can the platform improve efficiency and customer satisfaction?

## Dataset
- **Source:** FoodHub order data
- **Features include:** order ID, customer ID, restaurant name, cuisine type, cost of order, day of the week, rating, food preparation time, delivery time

## Skills & Tools
- **Language:** Python
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn
- **Techniques:** Exploratory Data Analysis (EDA), Descriptive Statistics, Data Visualization

## Key Findings
- Identified top-performing restaurants and most popular cuisine types
- Analyzed delivery time patterns across weekdays vs weekends
- Uncovered relationship between order cost, ratings, and delivery efficiency
- Provided actionable recommendations to enhance platform operations

## How to Run
1. Clone this repository
2. Install dependencies: `pip install pandas numpy matplotlib seaborn`
3. Open `FoodHub_Analysis.ipynb` in Jupyter Notebook or JupyterLab
4. Run all cells sequentially
